Simple Implementation

Acrobat Pros

  • Less overhead: Content from PMs, TAMS, engineering (AND LR).

User experience

  • Usable: Faster than the oft-outdated Help. No waiting for page load.

Registry configuration: HKCU/.../AVGeneral/SuperTips/

  • iTipSource:
    • 0 = none (disable feature)
    • 1 = local file (for end user content)
    • 2 = (Default) cloud file (for end user content)
    • 3 = cloud file (turn on IT Mode for IT-only content)
  • bTipAccessibility: Toggles whether the product should go into read-out-loud mode and read the tip text automatically. When set, tips must be manually dismissed, and tips are included in tab focus and F6 cycle.
