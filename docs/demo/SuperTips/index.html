Protected View

Recommendation: Enable Protected View for files from potentially unsafe locations.

Why?

  • PV is a secure, read-only mode that blocks most actions until you trust the document.
  • Once you trust a file, locked features becomes enabled.

IT Mode example

This UI item maps to iProtectedView which specifies whether to user Protected View never (default), for files from an untrusted location (recommended), or always.

  • HKCU setting: HKEY_CURRENT_USER\Software\Adobe\(product name)\(version)\TrustManager\iProtectedView
    • 0: (default) Disable Protected View.
    • 1: (recommended) Enable Protected View for unsafe locations only.
    • 2: Enable Protected View for all files.
  • Lockable setting: HKLM\SOFTWARE\Policies\Adobe\(product name)\(version)\FeatureLockdown\iProtectedView
