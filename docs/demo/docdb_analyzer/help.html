Project Database Analyzer Help
Create a .csv data file from AEM as follows:
  1. Open AEM Search.
  2. Specify a search path (your content directory)
  3. Select Search.
  4. Select the action Export results as CSV (selected fields).
  5. Select the Grid View icon to display possible CSV fields.
  6. Check ONLY the boxes shown in the image.
  7. In the Actions dropdown, select Export results as CSV (selected fields) You will receive an email with the results.
  8. Save the file locally.
  9. Upload the file to this web app.

Note that I can easily create additional tools using this same framework. If you need something else or find a bug, email brogers@adobe.com.

This version supports the specific fields below. For example, you must select the Loc Publish option and cannot choose the other localization option.

It's worth noting that AEM Search has bugs and that none of the 30-day fields work. Data may be inaccurate.
