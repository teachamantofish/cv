This app provides tools that leverage standard AEM Search CSV data exported from a workflow package. To add your own data, see these instructions. This app can also analyze live TOC data, and the loaded example uses the Adobe Connect Getting Started Guide, Training Guide, and Event Guide. TOC data collection requires 10 minutes of scripting. Contact brogers@adobe.com for changes, bugs reports, and feature requests.

Content management features

Actions that are faster here than in AEM:
  • Parse and analyze content data to augment AEM standard CSV data
  • Parse and analyze live TOC data AEM does not provide
  • Evaluate directory contents for cruft, age, EOL items
  • Evaluate the overall strategy and quality of titles, file names, and links to other guides
  • Find file copies and potential duplicates
  • Find files that were never published or are unpublished (so you can delete them)
  • Identify which files reside in different directories
  • Work in your browser with all the live links
  • Quickly find and work with the data you need: you'll only print and export what you see
  • For TOC data, identify which files are in each guide and evaluate content organization
  • When authoring in AEM, click less, copy less, paste less, swear less

Data view features


Find the data you need:
  • Change column visibility and page length
  • Sort on any column. Colors identify oddities to check
  • Use text, case-insensitive, and regular expressions to find and filter entries (.*, $, ^, etc.)

Batch action support


Use the search and filter options to show what you want. Select 1-N checkboxes for select specific files. Then you can:
  • Open all selected files in new browser tabs, including in author, preview, and live modes
  • Export a list of all selected files to a text file, Excel, or print
  • Email list of files to AEM web production team for them to delete, move, or otherwise manage
  • Email a list of review files (if they are published in preview mode)
  • Email a list of live content to anyone.
  • Create a GO URL file you can use for batch import in the GO URL tool.
  • Create a redirect file you can use for batch import into the Redirect tool.
  • Export all the TOC data you need to create an experience fragment from the selected files.
